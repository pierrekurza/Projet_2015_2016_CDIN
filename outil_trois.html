Github

Il s'agit d'un site qui permet héberger et de gérer des projets.
Il a également un logiciel complémentaire qui permet de rendre la gestion de données plus simple.

Application de Github

Ce site permet donc de gérer un projet, on peut crée des branches en plus de celle principale qui permet à chacun d'avancer dans sa partie et organiser le tout à la fin.
Il permet de voir l'avancement de chacun sans avoir à contacter les autres développeurs

Avantages & Inconvénients

Avantages

Voici les avantages de cet outil :

  • Gestion simple via les branches de projets.
  • Visibilité sur les changements/ajouts par les autres.
  • Site très rapide.
  • Application complémentaire permettant l'accès plus facile.

Inconvénients

Il y a tout de même quelques inconvénients :

  • Outil complexe à prendre en main.
  • Sans l'outil complémentaire, le portage sous Windows est complexe.
  • Le contenu est publique si on ne paye pas.
  • Beaucoup de commandes pour le partage de fichiers.
